# 📚 My Book Store Project

ยินดีต้อนรับสู่โปรเจกต์ร้านหนังสือออนไลน์ที่พัฒนาด้วย **Django Framework** และ **Bootstrap 5**

## 🚀 ฟีเจอร์ที่น่าสนใจ
- [x] ระบบแสดงรายการหนังสือทั้งหมด
- [x] ระบบสมัครสมาชิก (Register) พร้อมเก็บข้อมูลส่วนตัว (Profile)
- [x] ระบบเข้าสู่ระบบ (Login/Logout) ที่ปลอดภัย
- [ ] ระบบตะกร้าสินค้า (กำลังพัฒนา...)

## 🛠️ เทคโนโลยีที่ใช้
1. **Language:** Python 3.x
2. **Backend:** Django 4.2
3. **Database:** SQLite (เริ่มต้น)
4. **Frontend:** Bootstrap 5 (CSS Framework)

## 📦 วิธีการติดตั้งและรันโปรเจกต์

1. **สร้างและเปิดใช้งาน Virtual Environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # สำหรับ Mac/Linux
2. **ติดตั้ง Dependencies:**
   ```bash
   pip install -r requirements.txt  # ถ้ามีไฟล์ requirements
   ```
3. **Apply Migrations:**
   ```bash
   python manage.py migrate
   ```
4. **Run Server:**
   ```bash
   python manage.py runserver
   ```
